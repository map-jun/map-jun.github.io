### 현재 작업 폴더 확인 :

```R
getwd( )
```

### 작업 폴더 변경 :

맥 OS 

```R
setwd("/path/to/my/directory")
```

<br>

윈도우(Window) OS 

```R
setwd("c:/Documents/my/working/directory")
```
